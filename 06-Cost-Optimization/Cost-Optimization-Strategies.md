# Cost Optimization Strategies

The following strategies help reduce Azure operational costs.

## Compute

- Right-size Virtual Machines
- Stop unused VMs
- Avoid oversized SKUs

---

## Storage

- Use appropriate storage tiers
- Delete unused managed disks
- Remove unattached disks

---

## Networking

- Remove unused Public IPs
- Delete unused Load Balancers

---

## Monitoring

- Configure log retention policies
- Avoid collecting unnecessary logs

---

## Backup

- Configure appropriate backup retention
- Remove obsolete backups

---

## Governance

- Apply Tags
- Review Cost Analysis regularly
