# Security Implementation

The following security controls were implemented.

## Identity Security

- Microsoft Entra ID
- Azure RBAC
- Managed Identity

## Network Security

- Network Security Groups
- Subnet Isolation
- Least Privilege Access

## Secrets Management

- Azure Key Vault
- Secret Storage

## Governance

- Azure Policy
- Resource Locks
- Resource Tags

## Backup Security

- Recovery Services Vault
- Azure Backup

## Monitoring

- Azure Monitor
- Log Analytics
- Metric Alerts
