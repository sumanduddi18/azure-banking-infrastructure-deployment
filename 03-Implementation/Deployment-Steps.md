# Deployment Steps

The Azure Banking Infrastructure project was implemented in the following sequence.

## Phase 1
- Created Resource Group
- Created Virtual Network
- Created Subnets

## Phase 2
- Configured Network Security Groups
- Associated NSGs with Subnets

## Phase 3
- Deployed Windows Server Virtual Machine
- Configured Public IP
- Attached Managed Disk

## Phase 4
- Enabled Boot Diagnostics
- Configured Managed Identity

## Phase 5
- Created Azure Key Vault
- Stored Secrets

## Phase 6
- Configured Azure Monitor
- Created Log Analytics Workspace
- Enabled VM Insights

## Phase 7
- Created Metric Alerts
- Created Action Groups

## Phase 8
- Configured Cost Management

## Phase 9
- Configured Azure Backup
- Created Recovery Services Vault

## Phase 10
- Implemented Governance
- Applied Resource Locks
- Applied Tags
- Assigned Azure Policies

## Phase 11
- Validated Deployment
- Verified Resources
- Documented Architecture
