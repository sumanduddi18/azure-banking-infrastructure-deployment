# Azure Resources

The following Azure resources were deployed during the project.

| Resource | Purpose |
|----------|---------|
| Resource Group | Logical container for Azure resources |
| Virtual Network | Network isolation |
| Subnets | Separate Web and Database tiers |
| Network Security Groups | Traffic filtering |
| Public IP Address | Internet connectivity |
| Windows Server VM | Web server |
| Managed Disk | Persistent VM storage |
| Azure Key Vault | Secret management |
| Azure Monitor | Resource monitoring |
| Log Analytics Workspace | Centralized log collection |
| Metric Alerts | Performance monitoring |
| Action Group | Alert notifications |
| Recovery Services Vault | Backup management |
| Azure Backup | VM protection |
| Resource Locks | Prevent accidental deletion |
| Azure Policy | Governance and compliance |
| Resource Tags | Resource organization |
