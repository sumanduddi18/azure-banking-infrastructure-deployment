# Scenario Based Questions

## Scenario 1

Users cannot access the website.

What would you check?

- VM
- NSG
- Public IP
- IIS
- Azure Monitor

---

## Scenario 2

Developer accidentally deleted the VM.

How would you recover?

- Recovery Services Vault
- Azure Backup
- Restore VM

---

## Scenario 3

Application cannot access database.

Possible causes:

- NSG Rules
- Routing
- Windows Firewall
- Application Configuration

---

## Scenario 4

CPU utilization reaches 100%.

What actions would you take?

- Azure Monitor
- Alerts
- Performance Metrics
- Scale VM

---

## Scenario 5

Secrets exposed inside application.

Solution:

- Azure Key Vault
- Managed Identity

---

## Scenario 6

Production resources accidentally deleted.

Solution:

- Resource Locks
- Azure Policy

---

## Scenario 7

Unexpected Azure cost increase.

Solution:

- Azure Cost Analysis
- Advisor
- Stop unused resources
