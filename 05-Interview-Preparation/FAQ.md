# Frequently Asked Questions

## Why did you build this project?

To gain hands-on experience implementing a production-style Azure infrastructure covering networking, security, monitoring, governance, backup, and cost optimization.

---

## Which Azure services were used?

- Azure Resource Manager
- Virtual Network
- Network Security Groups
- Virtual Machine
- Managed Identity
- Azure Key Vault
- Azure Monitor
- Log Analytics
- Azure Backup
- Azure Policy
- Resource Locks
- Resource Tags

---

## What was the biggest learning?

Understanding how individual Azure services work together to build a secure and manageable enterprise cloud environment.

---

## Which part was most challenging?

Implementing governance controls and configuring monitoring and backup correctly.

---

## Which Azure certifications support this project?

- AZ-104 Azure Administrator Associate
- AZ-500 Azure Security Engineer Associate
