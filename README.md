## 📌 Project Overview

This project demonstrates the end-to-end deployment of a secure banking infrastructure on Microsoft Azure using production-oriented best practices.

The objective was not only to deploy Azure resources but also to implement identity management, network security, monitoring, backup, governance, cost optimization, and operational controls similar to those used in enterprise environments.

---

## 🎯 Project Objectives

- Deploy secure Azure infrastructure
- Implement network segmentation
- Configure Identity & Access Management (IAM)
- Secure secrets using Azure Key Vault
- Monitor infrastructure using Azure Monitor & Log Analytics
- Configure Backup & Recovery
- Apply Azure Policies and Resource Locks
- Optimize Azure cost and governance
