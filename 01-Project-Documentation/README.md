# 📘 Project Documentation

This folder contains the complete documentation for the Azure Banking Infrastructure Deployment project.

The project was designed to simulate a production-style Azure environment implementing infrastructure, networking, security, monitoring, governance, backup, and operational best practices.

## Project Highlights

- Production-style Azure deployment
- Secure networking using VNets, Subnets and NSGs
- Identity and access management using Microsoft Entra ID and RBAC
- Azure Monitor with Log Analytics and Alerts
- Azure Backup using Recovery Services Vault
- Azure Key Vault for secrets management
- Governance using Azure Policy, Tags and Resource Locks
- Cost optimization and operational best practices

---

## Outcome

The project demonstrates practical Azure Administration and Azure Security concepts aligned with enterprise cloud environments.
